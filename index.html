Giggie

A tool for creating, accepting, completing, and commenting on job/gig listings. Helps workers create alternative resumes with real objects as representatives of accrued experience; and helps clients find help with small but skill-heavy jobs without a long and complicated search and interview process.

Built using MongoDB, Express, React, and Node (MERN).

  • Created: December 2017
  • Other Links: Github

Goal-And-Get-It

Lets users create goals, assigns point values based on difficulty for each goal, update them, delete them, and earn points by completing them.

Made using HTML, CSS Bootstrap, jQuery, several different NPM packages including Sequelize and Express, and Firebase authentication, deployed on Heroku.

  • Created: October 2017
  • Other Links: Github

SenaTweet

Aggregates information about US senators from a few different APIs in order to allow side by side comparison of political activity and social media activity for each senator.

Made using HTML, CSS Bootstrap, jQuery, and Firebase authentication and deployment.

  • Created: August 2017
  • Other Links: Github
